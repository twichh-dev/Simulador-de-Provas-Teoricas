# Simulado DETRAN - Python & SQL Project

Este projeto foi desenvolvido com o objetivo de aprofundar conhecimentos técnicos em integração de software, manipulação de bancos de dados e criação de interfaces gráficas. Trata-se de um sistema de simulado para as provas do DETRAN, projetado para oferecer uma experiência prática de estudo e desenvolvimento.

## 🛠️ Tecnologias e Linguagens

O foco principal do desenvolvimento foi a aplicação prática das seguintes linguagens:

*   **Python:** Lógica de aplicação e interface gráfica (utilizando `tkinter` e `ttk`).
*   **SQL (SQLite):** Estruturação do banco de dados, armazenamento e recuperação dinâmica de questões.

---

## 📋 Características do Projeto

O sistema simula uma prova real do DETRAN, selecionando aleatoriamente questões de um banco de dados robusto. A distribuição das 60 questões segue o seguinte padrão:

| Categoria | Quantidade |
| :--- | :---: |
| Legislação | 9 |
| Direção Defensiva | 9 |
| Mecânica | 4 |
| Primeiros Socorros | 4 |
| Meio Ambiente | 4 |

### Objetivos Técnicos Alcançados:
*   **Integração Full-stack:** Conexão bem-sucedida entre o frontend em Python e o backend em SQL.
*   **Algoritmo de Aleatoriedade:** Implementação de sorteio de questões para garantir que nenhum simulado seja igual ao anterior.
*   **Gerenciamento de Estado:** Controle de progresso do usuário e correção automática ao final do teste.
